# Лабиринт сокровищ
Проект 1
## Что это за проект?
Представьте: вы — отважный исследователь в древнем лабиринте, 
полном хитрых коридоров, скрытых сокровищ и загадок, которые 
проверят вашу смекалку. 
Это не просто игра — это ваша собственная текстовая adventure 
игра "Лабиринт сокровищ", где вы программируете правила. 
Игрок (или вы сами в консоли) блуждает по комнатам, собирает артефакты 
в инвентарь, разгадывает ребусы и охотится за главным призом. 

Всё на чистом Python, в консольном стиле — никаких излишеств, 
только логика и код. Начнёте с простого туннеля в одном файле, 
а закончите хитрой сетью функций, где каждый поворот — это тест на мастерство.

```

```

## Игровые команды
| Команда | Действие |
|--------|--------|
| `go <direction>` | Перейти в указанном направлении (north, south, east, west) |
| `look` | Осмотреть текущую комнату |
| `take <item>` | Поднять предмет |
| `use <item>` | Использовать предмет из инвентаря |
| `inventory` | Посмотреть содержимое инвентаря |
| `solve` | Попробовать решить загадку |
| `help` | Показать список доступных команд |
| `quit` | Выйти из игры |
